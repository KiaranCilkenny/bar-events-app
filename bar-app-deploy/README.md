# Bar Events App

A React app for discovering sports bars, fan bars, and events.

## Deploy to Vercel

### Option 1: Vercel CLI (Recommended)
1. Install Vercel CLI: `npm i -g vercel`
2. Run: `vercel`
3. Follow prompts (press Enter for defaults)
4. Get your live URL!

### Option 2: Vercel Dashboard
1. Go to https://vercel.com
2. Sign up/login (GitHub recommended)
3. Click "Add New..." â†’ "Project"
4. Import this folder
5. Click "Deploy"

### Option 3: GitHub + Vercel (Best for updates)
1. Create GitHub repo
2. Push this code to repo
3. Connect repo to Vercel
4. Auto-deploys on every push!
